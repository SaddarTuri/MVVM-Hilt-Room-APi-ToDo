# 📱 MVVM-Hilt-Room-API App

A modern Android application built using **MVVM architecture**, demonstrating clean code practices and integration of **Hilt, Room, and REST APIs**.

---

##  Features
-  MVVM (Model-View-ViewModel) Architecture  
-  Dependency Injection with Hilt  
-  Local Database using Room  
-  API Integration (RESTful services)  
-  LiveData / State Management  
-  Clean and scalable project structure
